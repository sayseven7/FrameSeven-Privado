SP DRAGON SCAN

O SP DRAGON SCAN é um programa em Python que permite a verificação de portas abertas em um determinado endereço IP ou nome de domínio. Com ele, é possível identificar quais portas estão disponíveis e acessíveis, o que pode ser útil para testes de segurança ou para solucionar problemas de conexão.

BFTP

permite testar a segurança de servidores FTP. Com ele, é possível realizar ataques de força bruta, testando diversas combinações de usuário e senha até encontrar uma combinação válida e assim obter acesso ao servidor FTP.

BSSH

O Brutar SSH é um programa em Python que permite testar a segurança de servidores SSH. Com ele, é possível realizar ataques de força bruta, testando diversas combinações de usuário e senha até encontrar uma combinação válida e assim obter acesso ao servidor SSH.

CSQLI

programa em Python que permite testar a segurança de websites verificando vulnerabilidades de injeção de SQL (SQLi). Com ele, é possível realizar testes em URLs para verificar se existem vulnerabilidades de SQLi e assim prevenir ataques de hackers mal intencionados.

SQLS

O Sqlmap_Seven é um programa em Python que permite explorar vulnerabilidades de injeção de SQL (SQLi) em um website. Com ele, é possível testar uma URL em busca de vulnerabilidades e, se encontrado, o programa pode realizar diversas operações no banco de dados do site, como exibir informações confidenciais, criar novos usuários ou até mesmo apagar dados do banco.

SNMAP

O Nmap_seven é um programa em Python que permite automatizar a execução do Nmap, uma ferramenta de mapeamento de redes. Com ele, é possível executar diversos testes de segurança em uma rede, como a detecção de portas abertas, serviços em execução e até mesmo vulnerabilidades conhecidas.

HAWK BASILISC

O Buscar por SubDominios é um programa em Python que permite encontrar subdomínios de um determinado domínio. Com ele, é possível descobrir subdomínios ocultos de um site, o que pode ser útil para testes de segurança ou para mapear a infraestrutura de um site.

SSMPT

O Enumerar smtp é um programa em Python que permite enumerar usuários válidos em um servidor SMTP (Simple Mail Transfer Protocol). Com ele, é possível testar diversas combinações de usuários e senhas para encontrar credenciais válidas e, assim, acessar a conta de e-mail de um usuário.
